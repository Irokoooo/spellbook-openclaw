# 🧙 SpellBook Agent 从零到跑完全指南

欢迎！这篇教程会带你 **一步一步** 把 SpellBook 本地 Agent 跑起来。放心，就算你电脑上什么都没有，也能在几分钟内搞定。

---

## 📦 整体流程（一张图看懂）

```
下载安装包 → 运行安装 → 注册 Agent → 拿到config.toml → 启动 → 完成！
```

整个过程：**① 一键下载安装** → **② 点几下注册** → **③ Agent 就跑起来了**

---

## 第一步：下载一键安装包

1. 打开 SpellBook 网页 → 进入右上角 **设置** 页面
2. 找到「**一键安装 Agent**」卡片
3. 点击 **下载安装包 (16MB)** 按钮
4. 你会得到一个 `SpellBook_Agent_Setup.exe`

> 💡 这个安装包自带 Python 环境、所有依赖项、龙虾（OpenClaw），**不需要你自己装任何东西**。

---

## 第二步：运行安装程序

1. 双击刚下载的 `SpellBook_Agent_Setup.exe`
2. 按照安装向导操作（一路下一步就行）
3. 安装完成后，桌面会出现一个 🪄 SpellBook Agent 文件夹

> ⚠️ 如果 Windows 弹出 "Windows 已保护你的电脑"，点击「更多信息」→「仍要运行」即可。

---

## 第三步：在网页上注册你的 Agent

回到 SpellBook 网页的设置页面，找到「**本地 Agent**」卡片：

1. 在输入框里给你的 Agent 取个名字，比如「办公室电脑」或「笔记本」
2. 点击 **注册并下载配置**
3. 会自动下载一个 `config.toml` 文件

> 📝 `config.toml` 就是 Agent 的身份证，告诉它你是谁、连什么服务器。

---

## 第四步：放置配置文件

把刚下载的 `config.toml` 放到 Agent 目录里：

- 打开桌面的 `SpellBook Agent` 文件夹
- 把 `config.toml` 拖进去（覆盖已有的）

---

## 第五步：配置 AI 模型

在设置页面的「**龙虾（OpenClaw）配置**」卡片中：

1. 填入你的 **API Key**（DeepSeek / OpenAI / 通义千问 等）
2. 如使用非 DeepSeek 的模型，填入 **Base URL**（可选）
3. 点击 **下载配置文件 (.env)**
4. 把下载的 `.env` 文件放到：
   ```
   C:\Users\你的用户名\.openclaw\.env
   ```

> 🔑 如果没有 API Key？去 [DeepSeek 开放平台](https://platform.deepseek.com) 注册一个，新用户有免费额度。

---

## 第六步：启动 Agent！

打开桌面的 `SpellBook Agent` 文件夹，双击运行：

```
start_agent.bat
```

如果一切正常，你会看到一个命令行窗口，显示：
```
[INFO] Agent 已启动
[INFO] 心跳已发送
```

回到 SpellBook 网页设置页，你的 Agent 状态会从「离线」变成「**在线** 🟢」！

---

## 🎉 接下来就可以用了

回到聊天页面，输入任务，Agent 会自动拉取并执行！

比如试试：
- 「帮我爬取佛山最近的楼盘开盘数据，输出为 md 文件」
- 「帮我分析这份 Excel 表格」
- 「搜索一下最新的 AI 行业新闻并总结」

---

## ❓ 常见问题

| 问题 | 解决方法 |
|------|----------|
| Agent 显示离线 | 检查 config.toml 是否正确放在 Agent 文件夹里 |
| 提示 API Key 错误 | 检查 `.env` 文件是否放在正确位置，Key 是否有效 |
| 安装包被拦截 | 点击「更多信息」→「仍要运行」 |
| 之前装过龙虾 | 安装包会自动检测，不会重复安装 |

---

## 🔄 顺序总结

> **先下载安装包** → **运行安装** → **注册 Agent（拿到 config.toml）** → **配置 AI 模型** → **启动 Agent**

不需要纠结"谁先谁后"——安装包只管环境，注册只管身份，两者独立。只要把 config.toml 放到 Agent 目录，一切就通了 🎯
