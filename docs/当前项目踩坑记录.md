## 2026-06-12 🔴 API Key 泄露事件

- **泄露位置**：.claude/settings.json 中硬编码了 DeepSeek API Key (sk-d02b...)
- **原因**：.claude/ 目录未被 .gitignore 忽略，且 Claude 的权限配置中包含了真实 API Key
- **修复动作**：
  1. 替换 .claude/settings.json 中的真实 key 为 YOUR_DEEPSEEK_API_KEY 占位符
  2. .gitignore 新增 .claude/ 规则
  3. git commit --amend 重写历史，移除 .claude/ 文件
  4. git push --force 覆盖远程
- **用户需完成**：在 DeepSeek 控制台删除泄露的 key，生成新 key
- **教训**：任何包含 API key/secret 的文件都应加入 .gitignore；本地 AI 助手配置文件尤其危险
